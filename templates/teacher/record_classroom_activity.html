When to create an activity

Use this form when you want a named record — blackboard content, verbal instructions, or a shared Google Drive link. If you only need to enter scores, use Enter Activity Grades instead.

  1. 1 Create the activity — title, subject, period, and instructions
  2. 2 Enter activity grades — score each student on this activity
  3. 3 Scores roll up into draft period CA on the MoE grade sheet

For worksheets students submit online, use Drop activity files on the Activities tab instead. MoE CA + exam totals are entered separately on the Period Grade Sheet.
